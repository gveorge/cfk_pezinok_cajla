# Chelsea Website Design Analysis

## Layout Structure
- Horizontal category tabs at top (Senior, U21s, U19s, FA Youth Cup, U18s, On Loan, Staff)
- Player cards displayed in grid below the tabs
- Each player card shows:
  - Large player photo
  - Jersey number (large background)
  - First name (smaller)
  - Last name (large, bold)
  - Position (smaller text)

## Design Elements
- Clean, modern card-based layout
- Grid system (3 columns on desktop)
- Cards have white background with player photos
- Jersey numbers are subtle background elements
- Category tabs are simple text links that switch content without page reload

## Implementation for CFK Pezinok Cajla
- Use horizontal tabs for categories (U8, U9, U10, U11, U13, U15, A mužstvo)
- Default to A mužstvo tab
- Show player cards only for A mužstvo

- Keep everything on one page with tab switching
